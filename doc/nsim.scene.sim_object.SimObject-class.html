Parameters:
  • name (str) - An identifier for the SimObject. This should be castable to str.
  • orientation (ndarray or bool) - Orientation of the object. ndarray is interpreted as direction of orientation relative to the scene's positive z-axis. If True a random rotation is created. If False, no orientation. Default=False
  • points (arraylike) - Coordinates forming this object, expressed in the local coordinate system. None if no points, [0,0,0] represents self.position in the local coordinate system. Default=None
  • position (array like) - Position in scene (x,y,z). Default=[0,0,0]
  • sample_rate (float) - The sample rate in Hertz. Default=16000.0
