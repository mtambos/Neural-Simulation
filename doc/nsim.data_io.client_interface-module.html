Module client_interface

source code

data interface

This interface provides an (qt) event based interface to aquire and buffer data from a (potentially remote) data source (abstracted in another class). Incomming data is buffered until a desired amount is stored. The stored package is then passed on.

copy src[src_slc] to dst[dst_slc]

Helper function to copy an array slice from an source array src into a destination array dst. The slices (src_slc and dst_slc) must agree in size and define what slice will be copied where.
