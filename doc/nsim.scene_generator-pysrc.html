 29  """tool to generate a scene archive for the Neural Simulation 
 30   
 31  This module provides a GUI based approach to scene building. A initial scene can 
 32  be configured and saved as a loadable archive. The loadable archives are HDF5 
 33  containers with a predefined structure, as explained here: 
 34   
 35  SCENE ARCHIVE (.sca) - HDF5 container 
 36      __TYPE__ : string node 
 37          all groups must contain a '__TYPE__' node identifying its purpose as a  
 38          string. available identifiers are: 
 39              SCENE_ARCHIVE : for the root node 
 40              CONFIG        : for the NS config group 
 41              NEURON_DATA   : for the neuron data group 
 42              +NeuronData   : for NeuronData nodes 
 43              SCENE         : for the scene group 
 44              +SimObject    : for SimObject nodes 
 45      __CLASS__ : string node 
 46          various groups define the parameters set to instantiate a class. this 
 47          string holds th name of the class. meaning varies for context. 
 48       
 49      The basic structure for a scene archive is: 
 50           
 51          ROOT 
 52           |- __TYPE__ = 'SCENE_ARCHIVE' 
 53           |- CONFIG 
 54           |   |- __TYPE__ = 'CONFIG' 
 55           |   |- frame_size = 1024 
 56           |   |- sample_rate = 16000.0 
 57           |   |- [...] 
 58           |- NEURON_DATA 
 59           |   |- __TYPE__ = 'NEURON_DATA' 
 60           |   |- NDATA00 
 61           |   |   |- __TYPE__ = 'NeuronData' 
 62           |   |   |- __CLASS__ = '<neurondata subclass name>' 
 63           |   |   |- [...] 
 64           |   |- NDATA01 
 65           |   |   |- __TYPE__ = 'NeuronData' 
 66           |   |   |- __CLASS__ = '<neurondata subclass name>' 
 67           |   |   |- [...] 
 68           |   |- [...] 
 69           |- SCENE 
 70               |- __TYPE__ = 'SCENE' 
 71               |- SIMOBJ00 
 72               |   |- __TYPE__ = 'SimObject' 
 73               |   |- __CLASS__ = '<simobject subclass name>' 
 74               |   |- [...] 
 75               |- SIMOBJ01 
 76               |   |- __TYPE__ = 'SimObject' 
 77               |   |- __CLASS__ = '<simobject subclass name>' 
 78               |   |- [...] 
 79               |- [...] 
 80       
 81      All nodes are taged with at __TYPE__ tag and when a node specifies an 
 82      instanciateable class, a __CLASS__ tag is additionall given. 
 83       
 84      [t.b.c]
