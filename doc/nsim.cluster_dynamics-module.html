cluster_process(single_rates, nsmpls, srate, o2rate=5.0, o3rate=1.0)

source code 
produce spike trains for N units with overlap rates.
Parameters:
  • single_rates (list) - List of firing rates (in Hz) for the single units in the cluster
  • nsmpls (int) - Length of the simulated events in samples.
  • srate (float) - Sample rate (in Hz).
  • o2rate (float) - Overlap of 2 events, rate in Hertz. If 0.0 do not put overlaps. Default=5.0
  • o3rate (float) - Overlap of 3 events, rate in Hertz. If 0.0 do not put overlaps. Default=1.0
